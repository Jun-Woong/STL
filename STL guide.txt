신입생을 대상으로한 STL 프로그래밍 프로젝트 안내입니다.

이 프로젝트는 연구실 신입생을 대상으로 진행하는 과제로, 연구실 졸업생이신 이상필 박사님께서 고안하셨습니다.

연구실에서 사용하는 대부분의 시뮬레이터가 C++로 구현되어 있습니다. 본 프로젝트는 c++을 보다 능숙하게 다룰 수 있도록 STL (standard template library) container를 사용해보는데 목적이 있습니다.

기본적인 C++ 관련 내용들과 (OOP, class, 상속, template) STL container들의 구조 및 특징을 개인적으로 찾아서 공부하시고 아래 내용을 진행하시면 되겠습니다.

될 수 있는한 혼자의 힘으로 모든 프로젝트를 완성시켜 보는 것을 추천드립니다. 



1.  과제 진행 스케줄

    1~2주차 – Sequence container 구현
    -       Vector
    -       Deque : double-ended queue
    -       List : linked list

    3주차 – Adaptor container 구현
    -       Stack : LIFO stack
    -       Queue : FIFO queue
    -       Priority queue

    4주차 – Associative Container 구현
    -       Set
    -       Map

2.  개발환경 구축

    i.      Virtual machine 및 linux ubuntu 설치

    ii.     Programing 환경 조성 (IDE 설치, ssh/ftp 설정)

    iii.    Makefile을 사용해 make 가능한 환경 조성

3.  프로젝트 진행 과정

    i.      makefile 작성 (tmeplate을 참고하여 자유롭게 작성)

    ii.     해당 주차 STL container에 대한 학습 및 이해

    iii.     main.cc에 구현되어 있는 STL function을 *.h에 구현 (1주차는 -def.h 제공)

    iv.    출력 결과를 result.log와 비교

    v.     매 주 한 project를 진행하여 3주 동안 모든 header 완성

6.  참고 문서

    i.      STL : http://www.cplusplus.com/reference/stl/
